# Правовой Фонд - Платформа Юридической Фирмы

Это готовый к работе (production-ready) Next.js проект для юридической фирмы. Проект поддерживает редактирование контента через простые JSON файлы и имеет современный SEO-оптимизированный дизайн.

## Требования
- Node.js (рекомендуется v18 и выше)
- npm или bun (проект использует bun/npm)

## Как запустить проект

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите сервер для разработки:
   ```bash
   npm run dev
   ```

3. Для сборки в продакшен:
   ```bash
   npm run build
   npm run start
   ```

## Как редактировать контент

Весь контент сайта хранится в виде JSON-файлов в папке `cms/`. Вы можете редактировать тексты, ссылки и данные прямо в них, и изменения применятся на сайте. Отдельного дашборда для администратора нет, что обеспечивает высокую скорость работы и простоту управления без баз данных.

### Как добавить адвоката

1. Откройте файл `cms/lawyers.json`.
2. Добавьте новый объект в массив (скопируйте существующий). Пример:
   ```json
   {
     "id": "4",
     "name": "Имя Фамилия",
     "photo": "/images/lawyer-new.jpg",
     "specialization": "Ваша специализация",
     "description": "Краткое описание.",
     "education": "Ваше образование",
     "experience": "Ваш опыт работы",
     "slug": "imya-familiya"
   }
   ```
3. Фотографию `lawyer-new.jpg` положите в папку `public/images/`.

### Как добавить услугу

1. Откройте файл `cms/services.json`.
2. Добавьте новый объект в массив услуг. Уникальный параметр `slug` определяет URL (например, `/services/vasha-usluga`).

### Как добавить статью

1. Откройте файл `cms/articles.json`.
2. Добавьте новую статью. Параметр `slug` определяет URL. Статья автоматически появится на странице "Статьи".

### Изменение контактной информации и текстов главной страницы

- `cms/contact.json`: содержит название компании, телефон, email, адрес и ссылки на мессенджеры.
- `cms/home.json`: содержит все тексты для главной страницы (Hero section, преимущества, заголовки секций).
